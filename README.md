# Emergency-Vehicle-Detection-and-Signal-Triggering
This project presents a real-time, multi-modal Emergency Vehicle Detection System integrated with the SUMO (Simulation of Urban Mobility) traffic simulator to dynamically control traffic signals and create a green corridor for emergency vehicles.
